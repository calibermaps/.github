# CaliberMaps

CaliberMaps builds **VectorTopo**, a field-first mapping platform for land,
claims, terrain, and the outdoors.

VectorTopo brings authoritative public data into one fast map — mining claims,
public land and parcels, hunting and fishing regulations, geology, backcountry,
weather, and high-resolution terrain — on the web and on iOS and Android.

Learn more at [vectortopo.com](https://vectortopo.com). Currently in private beta.
